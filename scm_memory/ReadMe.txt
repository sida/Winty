========================================================================
    ダイナミック リンク ライブラリ : scm_memory プロジェクトの概要
========================================================================

AppWizard はユーザー用に scm_memory DLL を作成しました。  
このファイルには scm_memory アプリケーションを構成している
各ファイルの概要が含まれています。


scm_memory.vcproj
    これはアプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    ファイルが生成された Visual C++ のバージョン情報が含まれています。また、 
    アプリケーション ウィザードで選択したプラットフォーム、構成およびプロジェクト機能に関する
    情報も含まれています。

scm_memory.cpp
    これはメインの DLL ソース ファイルです。

	この DLL は作成時にシンボルをエクスポートしません。そのため、 
	ビルド時に .lib ファイルが生成されません。このプロジェクトが、
	ほかのプロジェクトにプロジェクト依存になることを望む場合は、次のことを行ってください。 
	エクスポート ライブラリが作成されるように、シンボルをエクスポートするコードを DLL から追加するか、 
	または、プロジェクトのプロパティ ページのダイアログ ボックスを開き、リンカ フォルダの全般プロパティ ページで、
	入力ライブラリ プロパティを無視するように
	設定します。

StdAfx.h, StdAfx.cpp
